Il menu **Home** di Arm si compone delle seguenti voci:  

* **Licenses**
* **Company**
* **Connections**
* **Applications**
* **Database Management**
* **Custom Data Management**
* **Context**
* **Themes**

Per ciascuna di queste voci è dedicato un menu con all'interno la spiegazione della funzionalità delle singole form.
